# 🥋 Hajime Dojo

Веб-приложение для школы карате.  
Проект создан для управления учениками, тренерами, посещаемостью, новостями и статусом оплаты.

## Возможности проекта

- личный кабинет ученика
- кабинет тренера
- админ-панель Django
- учёт посещаемости тренировок
- отображение статистики посещаемости
- новости школы
- статус оплаты ученика
- роли пользователей:
  - ученик
  - тренер
  - администратор

## Технологии

- Python
- Django
- HTML
- CSS
- SQLite — для локальной разработки
- PostgreSQL — планируется для продакшен-версии

## 📋 Основные модули

### 👥 Пользователи системы и роли

- 🥋 **Ученик** — доступ к личному кабинету, расписанию и истории
- 🏅 **Тренер** — ведение журнала посещаемости и отметки об оплате
- ⚙️ **Администратор** — полное управление системой и новостями

---

### 📊 Профили учеников

- 📅 Дата рождения  
- 🟤 Пояс (цвет / уровень)  
- ⚖️ Весовая категория  
- 🏆 Спортивная категория  
- 🏅 Количество соревнований  
- 🎖️ Количество побед  

---

### 📝 Журнал посещаемости (Attendance)

- ✅ Тренер отмечает присутствие учеников  
- 👁️ Ученик видит историю посещаемости  
- 📈 Автоматически считается статистика посещений  

---

### 💰 Статус оплаты (Payments)

- 💵 Тренер может отметить, оплачено ли обучение  
- 📱 Ученик видит текущий статус оплаты в личном кабинете  

---

### 📰 Новости школы (News)

- ✍️ Публикация новостей через админку  
- 📋 Отображение списка новостей на главной  
- 📄 Отдельная страница для каждой новости
